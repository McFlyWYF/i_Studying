# 爱学习
刘鼎杯作品(硬件与软件的结合)
### 基于传感器与图像识别的智能自习室
* 软件主要功能 
  * 自习室查找
  * 好友聊天
  * 室外导航
  * 招聘信息
  * 失物招领
  * 校园新闻
* 硬件功能
  * 照明灯的智能控制
  * 温湿度的自动控制
  * 教室设备管理
  * 烟雾火情报警系统
